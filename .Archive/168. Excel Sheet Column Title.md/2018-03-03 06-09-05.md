# Description
Given a positive integer, return its corresponding column title as appear in an Excel sheet.
For example:

```
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
```

## 1.
```python
def convertToTitle(self, n):
    string = ""
    while n != 0:
        if not n % 26:
            string = 'Z' + string
            n = (n - 26) // 26
        else:
            string = chr(n%26 + 64) + string
            n = (n - n%26) // 26
    return string
```
