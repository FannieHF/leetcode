##1. brute - count() + **&**
```python
def judgeCircle(self, moves):
    """
    :type moves: str
    :rtype: bool
    """
    return ((moves.count("R") == moves.count("L")) & (moves.count("U")==moves.count("D")))
```

## 2. brute - count() + **and**
```python
def judgeCircle(self, moves):
    return moves.count("R") == moves.count("L") and moves.count("U")==moves.count("D")
```
