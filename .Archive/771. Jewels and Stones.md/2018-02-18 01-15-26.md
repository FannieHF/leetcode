ONE. submission 1 [accepted] [O(js)]
```python
def numJewelsInStones(self, J, S):
    """
    :type J: str
    :type S: str
    :rtype: int
    """
    sum = 0
    for s in S:
        if s in J:
            sum += 1
    return sum
```
