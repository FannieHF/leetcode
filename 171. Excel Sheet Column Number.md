# Description
Given a column title as appear in an Excel sheet, return its corresponding column number.
For example:
```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
```

##1.
```python
def titleToNumber(self, s):
    col = 0
    for letter in s:
        col = col*26 + ord(letter.upper()) - 64
    return col
```
